# FavRecipes
Android app showing food recipes from API [https://spoonacular.com/food-api](https://spoonacular.com/food-api)

Language: Kotlin

## Libraries used 

- Material Components
- Navigation Components
- Room Database
- Data Binding
- DataStore
- Retrofit
- Hilt
- Coil
- Gson
- Shimmer - `todkars`
- Jsoup
- and more...
